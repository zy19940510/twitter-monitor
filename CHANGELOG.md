## [1.0.0] - 2026-01-29

### Added
- Initial release
- LangGraph-based workflow engine
- Multi-provider LLM support (7 providers)
- CDP-based Twitter login and scraping
- Automatic login state management
- SQLite-based tweet deduplication
- Telegram push notifications
- Configurable environment variables
- Cross-platform support (macOS, Linux, Windows)
- Comprehensive documentation (English + Chinese)

### Features
- **Fetch Agent**: Browser automation via agent-browser
- **Analyse Agent**: AI-powered tweet analysis
- **Push Agent**: Telegram Bot API integration
- **Smart scrolling**: Configurable scroll count for tweet loading
- **Ad filtering**: Automatic filtering of promoted content
- **Login verification**: Auto-detect login status with error messages

### Configuration
- Chrome path configuration
- Chrome user data directory
- CDP port configuration
- Tweet analysis limits
- Display limits

- Log directory
